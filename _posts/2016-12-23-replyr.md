오늘 r-blogger.com에 흥미로운 패키지가 소개되었습니다.

`replyr`이란 이름의 패키지는 `dplyr`을 좀 더 쉽게 사용할 수 있게 해줍니다.

``` r
library(replyr)
head(Theoph) %>% replyr::letp(alias = list(c1 = "Wt", c2 = "Dose", c3 = "Total.Dose"), 
    expr = {. %>% mutate(c3 = c1 * c2)})
```
